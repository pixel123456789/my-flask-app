Site Map

This site map provides a comprehensive overview of the structure and functionality of the website. Use it to navigate and understand the various features available.

Website Structure

1. Home Page

The landing page of the site. It includes:

  • Hero Section: Introduction with a brief overview of Liam Kinnaird and his work.
  • Portfolio Gallery: Displays various projects with images and descriptions.
  • Contact Form: Allows visitors to send messages to Liam.

2. About Page

Provides information about Liam Kinnaird's background, skills, and career goals. Key sections include:

  • Introduction: Brief personal introduction and career aspirations.
  • Education: Details about current studies and future goals.

3. Portfolio Page

Showcases various projects completed by Liam. Each project includes:

  • Project Image: Visual representation of the work.
  • Description: Brief overview of the project and its features.

4. Contact Page

Allows users to get in touch with Liam. The form includes:

  • Name: Required field for the user's name.
  • Email: Required field for the user's email address.
  • Message: Field for the user to enter their message.

5. Reviews Page

Users can submit and view reviews. Features include:

  • Review Form: Submit a review with a star rating, name, order ID, and content.
  • Review Carousel: Displays submitted reviews in a rotating carousel.

6. Dashboard Page

Accessible to authenticated users for managing their information. Includes:

  • Profile Management: View and update personal details.
  • Additional Features: Based on user roles and permissions.

Statuses and Messages

1. Flash Messages

Temporary messages displayed after user actions. They provide feedback such as success or error notifications.

2. Error Messages

Displayed when there are issues with user input or server processes. They guide users on how to correct the issues.

How to Use the Website

1. Navigating the Site

Use the navigation bar at the top to access different sections of the website. Each link will take you to the respective page or section.

2. Submitting Reviews

Fill out the review form on the Reviews page with your name, order ID, review content, and star rating. Your review will be validated and added to the carousel upon successful submission.

3. Contacting Liam

Complete the contact form on the Contact page to send a message. Ensure all required fields are filled out before submission.

4. Using the Dashboard

Log in to access your dashboard. Manage your profile and interact with additional features provided based on your user role.
